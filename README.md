# new-repository-game-predictor
Ce projet de M2 à pour but d'utiliser l'api github pour prédire, à la semaine suivante, le nombre de nouveau répertoire créer pour un jeu spécifié.
Le projet utilise plusieurs modèles différent et effectue un réentrainement par semaine sur tout les modèles et il séléctionne le modèle pour effectuer les prédictions.
Des liens API sont disponibles pour engendrer manuellement l'entrainement, la collecte de donnée et les prédictions mais tout est automatisé.
